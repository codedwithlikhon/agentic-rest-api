## Agent Instructions

This document provides instructions for agents working with this codebase.

### Codebase Overview

This is a Flask API with the main application logic in `api.py`. The API specification is defined in an `openapi.yaml` file using the OpenAPI 3.1.0 standard.

### Development Workflow

1.  **Understand the Task:** Before making any changes, make sure you have a clear understanding of the task. If anything is unclear, ask for clarification.
2.  **Create a Plan:** Once you understand the task, create a plan of action. This should include the files you need to modify, the changes you need to make, and how you'll test your changes.
3.  **Implement the Changes:** Implement the changes according to your plan.
4.  **Test Your Changes:** Run any relevant tests to ensure that your changes are correct and don't break anything.
5.  **Submit Your Changes:** Once you're confident that your changes are correct, submit them for review.

### Pre-Commit Checks

Before submitting your changes, make sure to run the pre-commit checks. This includes running any relevant tests, requesting a code review, and recording your learnings.
